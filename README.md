# Outbound Calling Conversion Propensity Model

## Overview

This project demonstrates a Python-based customer conversion propensity scoring model designed to prioritise outbound calling campaigns for financial products including:

* CTF
* JISA
* ISA
* Over 50

The model analyses customer behavioural and account indicators to identify customers with the highest conversion potential.

---

## Business Problem

Call centre teams require prioritised customer lists to improve campaign efficiency, increase conversion rates, and focus on high-value opportunities.

This solution scores customers using:

* Maturity date proximity
* Active Direct Debit status
* Failed Direct Debit history
* Account value
* Online login activity
* Complaints history
* Product type

## Model Logic

The model applies rule-based predictive scoring logic to classify customers into:

* High Priority
* Medium Priority
* Low Priority

Customers with higher scores are prioritised for outbound calling campaigns.
