# Outbound Calling Conversion Propensity Model

## Overview

This project demonstrates a Python-based customer segmentation and conversion propensity model using behavioural analytics and CACI Fresco-style demographic segmentation for financial products including:

* CTF
* JISA
* ISA
* Over 50

The solution prioritises customers for outbound marketing campaigns and recommends the most suitable next-best-action based on customer behaviour, digital engagement, financial value, product maturity, Direct Debit status, and complaint history.

The objective is to identify customers with the highest conversion potential and support more targeted, efficient, and data-driven campaign activity.

---

## Business Problem

Call centre teams require prioritised customer lists to improve campaign efficiency, increase conversion rates, and focus on high-value opportunities.

This solution scores customers using:

* Maturity date proximity
* Active Direct Debit status
* Failed Direct Debit history
* Account value
* Online login activity
* Complaints history
* Product type

## Model Logic

The model applies rule-based predictive scoring logic to classify customers into:

* High Priority
* Medium Priority
* Low Priority

Customers with higher scores are prioritised for outbound calling campaigns.
